        ============================================================
        2. URL ROUTES CREATED (FRONTEND ONLY)
        ============================================================

        These URLs currently show frontend pages (NO backend functionality yet):

        1. /admin-login/                → Admin login page  
        2. /admin-dashboard/            → Dashboard page  
        3. /admin-doctors/              → Doctor listing page  
        4. /admin-doctor/add/           → Add doctor page  
        5. /admin-doctor/<id>/edit/     → Edit doctor page  
        6. /admin-doctor/<id>/view/     → View doctor page  
        7. /admin-doctor/<id>/delete/   → Delete doctor (temporary dummy)  
        8. /admin-patients/             → Patients list  
        9. /admin-patient/<id>/details/ → Patient details  
        10. /admin-appointments/        → Appointment list  
        11. /admin-feedback/            → Feedback page  
        12. /                           → Landing page  

        Backend team MUST connect the database to these endpoints later.
